# MorganaLM

A personal AI assistant that promotes mental health habits by tracking data (sleep, activity, moods) and providing affirmations via a mobile app by connecting with other tracking apps.

This is a testing prototype, so placeholder health data is used.

All written code is in the \lib directory. 

## Getting Started

Download the APK: https://drive.google.com/file/d/1SJY7t6vj-sBKjRUN1Ai20S4-gye6TZyP/view?usp=sharing

Alternatively, you can import this project in the Android Studio IDE.

It is recommended that this project ran with an emulator. This project was tested with an Pixel 3a XL emulator.
