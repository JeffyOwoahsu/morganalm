# MorganaLM

A personal AI assistant that promotes mental health habits by tracking data (sleep, activity, moods) and providing affirmations via a mobile app by connecting with other tracking apps.

This is a testing prototype, so placeholder health data is used.

All written code is in the \lib directory.
